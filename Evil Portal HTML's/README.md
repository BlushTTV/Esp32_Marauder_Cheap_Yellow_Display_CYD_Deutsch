## 🌍 Beschreibung
Das **Evil Portal** Modul des ESP32 Marauders ermöglicht es, eigene WLAN-Anmeldeseiten (Captive Portals) zu hosten. 

* **Zweck:** Demonstration von Phishing-Angriffen und Sensibilisierung von Nutzern für die Gefahren in öffentlichen WLAN-Netzwerken.
* **Inhalt:** In diesem Ordner findest du verschiedene Vorlagen (z. B. Google-Login, Starbucks, Standard-WiFi-Login), die für den ESP32 optimiert sind.

## 🛠 Installation
1. Wähle eine Vorlage aus dem Verzeichnis aus.
2. Kopiere die `index.html` in das Stammverzeichnis von deiner Marauder SD Karte.
3. Starte das Modul über das Marauder-Menü.
